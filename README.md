# Baseball CLI

A command-line application for baseball-related operations.

## Building

```bash
dotnet build
```

## Running

```bash
dotnet run -- [command] [options]
```

## Available Commands

- `--help, -h` - Show help message
- `--version, -v` - Show version information

## Development

This project uses .NET 6.0. Make sure you have the .NET SDK installed.

```bash
# Build the project
dotnet build

# Run tests (when added)
dotnet test

# Publish a release build
dotnet publish -c Release
```
